# 🌝맛점몇점<br>
## How to start
1. frontend<br>
`npm run build`
2. backend<br>
`npm start`
3. open browser<br>
`http://localhost:3000`

#### 📌음식점 등록
<p align="center">
  <img src="https://user-images.githubusercontent.com/55904021/209618954-5cda8f0a-12c2-4d3d-90b3-98f13e002717.png"/>
